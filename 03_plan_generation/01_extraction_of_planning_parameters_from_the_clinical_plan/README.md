# Extraction of planning parameters from the clinical plan

## Purpose
Extract the minimum set of clinically relevant parameters needed to reconstruct a baseline research plan.

## Inputs
- original clinical plan in ESAPI / PyESAPI

## Outputs
- parameter text files containing identifiers, isocentre, prescription, and optimisation objectives

## Included material
- `extract_plan_parameters.py` — sanitized parameter extraction example
- `parameters.example.txt` — example parameter file
